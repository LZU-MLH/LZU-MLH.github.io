JavaScript基础

1. 初识Javascript

$1.1 基础知识

​ JavaScript是一种运行在客户端的脚本语言

  • 脚本语言：不需要编译，运行过程由JS解释器逐行来解释执行
  • 产生的最初目的为表单动态验证，后来又可以做网页特效，服务器开发，桌面程序，APP，物联网，游戏开发
  • 浏览器组成
    • 渲染引擎：俗称内核，如Chrome浏览器的blink等
    • JS引擎，也称JS解释器，用来处理JS代码，对其处理后执行，如Chrome浏览器的V8

$1.2 JS组成

  • ECMA Script：规定了JS的编程语法和基础知识，是所有浏览器厂商共同遵循的一套JS语法工业标准
    • JavaScript：网景公司
    • Jscript：微软公司
  • DOM：页面文档对象模型
  • BOM：浏览器对象模型

$1.3 书写位置

  • 行内
  • 内嵌：<script>JS代码</script>
  • 外联：<script src="my.js"></script>

$1.4 注释

​ 同C语言：

  • 单行：//
  • 多行：/**/

$1.5 输入输出

  • 输入：prompt('输入内容')
  • 弹出警示框：alert('警示内容')
  • 控制台输出：console.log('日志内容')
    • 在浏览器：右键->检查->控制台

2. 变量

本质：储蓄在内存中申请的一块用来存放数据的空间

  • 赋值与初始化
var age;
age = 18;
console.log(age);

var name = 'Ahui'
console.log(name);
  • 命名规范：同C语言
  • 只声明，不赋值：输出undefined
  • 不生命，不赋值：报错
  • 不声明，只赋值：正常输出，会变为全局变量

3. 数据类型

  • JS是一种弱类型语言，变量数据类型是程序在运行过程中，根据等号右边的值才能确定

  • JS是动态语言，变量的数据类型可以任意变化

$3.1 数字型

  • 八进制以0开头，十六进制以0x开头
  • 最值alert(Number.MAX_VALUE) | alert(Number.MIN_VALUE)
  • 特殊值
    • Infinity：代表无穷大
    • -Infinity：代表无穷小
    • Nan=N：非数值，可用isNaN()方法验证

$3.2 字符串型

  • 可以是引号中的任意文本，可以使用单引号或双引号
  • 转义字符，同C语言
  • 字符串长度：str.length
  • 字符串拼接：+
    • aaa + bbb = aaabbb
    • aaa + 13 = aaa13

$3.3 布尔型

boolean分类：true和false

  • 在算数运算中，true当作1，false当作0

$3.4 未定义型

undifined：如果一个变量声明但未赋值，就是undifined，也可以直接将变量直接定义为undifined

  • 在字符串拼接中，当作普通字符串看
  • 在算数运算中，结果为NaN

$3.5 空值

null：即空值

  • 在字符串拼接中，当作普通字符串看
  • 在算数运算中，null当作0

$3.6 获取变量的数据类型

语法：typeof 变量名

5. 运算符

$5.1 算数运算符

  • 加减乘除取余，例：alert(5 / 10)：结果为0.5
  • 表达式与返回值：同C语言

$5.2 递增递减运算符

$5.3 比较运算符

<, >, <=, >=,

==, !=, ===

  • 18 == '18'，结果为true，因为等号默认转换数据类型
  • 18==='18'，判断左右两侧是否值和类型均相等

$5.4 逻辑运算符

与或非，即&&, ||, !

  • 逻辑与短路原理：表达式1 && 表达式2
    • 如果表达式1为真，则返回表达式2
    • 如果表达式1为假，则返回表达式1，并终止运算
  • 逻辑或短路原理：表达式1 || 表达式2
    • 如果表达式1为真，则返回表达式1
    • 如果表达式1为假，则返回表达式2，并终止运算

$5.5 逻辑运算符

=, +=, -=, *=, /=, %=

  • if, else if, else
  • 表达式? 选项1:选项2

	case value1: 语句1; break;

9. 函数

//声明函数
//方法一
function getSum(L, R)
{
    var sum = 0;
    for(var i = L; i <= R; i++)
        sum += i;
    return sum;
}
//方法二：函数表达式方法
var getSum = function(L, R)
{
    var sum = 0;
    for(var i = L; i <= R; i++)
        sum += i;
    return sum;
}
//调用函数
alert(getSum(10, 20));
//有形参和实参的区别
  • 形参和实参个数不匹配时
    • 实参个数多，则省略多余部分
    • 形参个数多，则多余形参定义为undefined
  • arguments伪数组，并不是真正意义上的数组
    • 具有数组的length属性
    • 按照索引的方式存储
    • 没有真正数组的一些方法，如pop()，push()
//求任意个数最大值
function getMax()
{
    var Max = arguments[0];
    for(var i = 1; i < arguments.length; i++)
        if(arguments[i] > Max)
            max = arguments[i];
    return Max;
}

10. 作用域

  • 全局作用域与局部作用域

    • 全局：整个script标签中，或是一个单独的文件中
    • 局部：函数内部

  • 块级作用域还未使用，即{}中的变量

  • 内部函数访问外部函数的内容，称作作用域链

11. JS预解析

JS引擎会把JS里面所有的var和function提升到当前作用域的最前面，然后从上往下执行代码

  • 分类
    • 变量预解析(变量提升)，提升变量声明，不提升赋值操作
    • 函数预解析(函数提升)，提升函数声明，不进行调用操作

12. JS对象

对象是由属性和方法组成的

  • 属性：事物的特性
  • 方法：事物的行为

$12.1 创建对象

//1. 利用字面量，即{}，键值对形式，逗号间隔，最后一个可以不使用,
var obj = {
    uname: 'Ahui',
    age: 18,
    sayHi: function() {
        console.log('Hi~');
    }
};
//2. 利用new Object，利用等号赋值的方法
var obj = new Object();
obj.uname = 'Ahui';
obj.sayHi = function() {
     console.log('Hi~');
}
//3. 利用构造函数的方法
//把对象里面一些相同的属性和方法抽象出来封装到函数中
function People(uname, age, sex) {
    this.uname = uname;
    this.age = age;
    this.sex = sex;
    this.sayHi = function() {
        console.log('Hi~');
    }
}
var obj = new People('Ahui', 18, 'Man')

使用方法：

  • 对象名.属性名：obj.uname，obj.sayHi()
  • 对象名.['属性名']：obj['age']

new执行过程

  • 在内存中创建一个空的对象
  • this指向刚才的空对象
  • 执行构造函数里的代码

遍历对象：

for (var k in obj)
{
    console.log(k);
    console.log(obj[k]);
}

13. 内置对象

  • JS中的对象分为三种：自定义对象，内置对象，浏览器对象

  • 内置对象就是JS语言自带的一些对象，提供了一些基本的功能，可理解为C语言库文件

$13.1 Math对象

//常用属性和方法
Math.PI
Math.floor() // 向下取整
Math.ceil()	 // 向上取整
Math.round() // 四舍五入取整
Math.abs()	 // 获得绝对值
Math.max()	 // 求最大值
Math.min()	 // 求最小值
Math.random()// 获得一个随机数，范围[0,1)

$13.2 日期对象

​ 是一个构造函数，必须使用new来创建日期对象

$13.2.1 基本使用

//如果没有参数，返回当前系统的当前时间
var date = new Date();
//参数常用写法：
//数字型
var date = new Date(2019, 10, 01);
//字符串型
var date = new Date('2019-10-1 8:8:8')

$13.2.2 格式化

var date = new Date();
console.log(date.getFullYear());
console.log(date.getMonth() + 1);//从0开始计数，所以加1
console.log(date.getDate());
console.log(date.getDay());//周日返回0，其余相同

//还有getHours()，getMinutes(), getSeconds()

//获得时间戳，即从1970年第一天距今的毫秒数
//方法1：
console.log(date.valueOf());
console.log(date.getTime());
//方法2：
var date1 = +new Date();
console.log(date1);
//方法3：
console.log(Date.now());

$13.3.1 数组创建和检验

var arr1 = new Arrar();//创建一个空的数组
var arr2 = new Arrar(2);//创建一个空的长度为2的数组
var arr1 = new Arrar(2, 3);//等价于[2, 3]

//检测是否为数组
console.log(arr3 instanceof Array);
console.log(Array.isArray(arr3));

$13.3.2 数组元素操作

$13.3.3 其他操作

  • 反转数组：arr.reverse()
  • 数组排序：arr.sort()
    • 按照大小比较升序排序：arr.sort(function(a, b) { return a - b;});
  • 数组索引方法
    • arr.indexOf()：返回第一个满足条件的索引号，没有返回-1
    • arr.lastIndexOf()：返回最后一个满足条件的索引号，没有返回-1
  • 转化为字符串：arr.toString()
  • 添加分隔符：arr.join('-')

$13.4 字符串对象

//基本包装类型，有String，NUmber，Boolean
var str = 'Andy';
//执行步骤
//1. 把简单数据类型包装为复杂数据类型
var tmp = new String('Andy');
//2. 把临时变量的值给str
str = tmp;
//3. 销毁临时变量
tmp = null;

$13.4.1 字符串不可变

​ 指的是里面的值不可变，虽然改变了内容，但实际上是地址变了，内存中开辟了一个新的空间

$13.4.2 其他操作

  • 查找字符：str.indexOf('查找字符', '起始位置')，其中第二个参数可以省略，返回值为字符所在位置

  • 根据位置返回字符:

    • str.charAt(位置)：返回对应位置字符
    • str.charCodeAt(位置)：返回对应位置字符的ASCII码
    • str[位置]：有兼容性要求

  • 连接多个字符串：concat('字符串1'，字符串2')，用+更简单

  • 字串：

    • substr(strat, length)
    • slice(start, end)
    • substring(start, end)

  • 替换字符串：replace('被替换字符', '替换为字符')，只会替换第一个匹配到的字符，返回替换后的字符串

  • 字符串转化为数组：str.split('分隔符')

14. 简单数据类型与复杂数据类型

  • 简单类型，又叫值类型，在存储时变量中存储的是值本身，如string, number, boolean, undefined, null，存储时放到栈里面
    • 直接寻址
  • 复杂类型，又叫引用类型，在存储时变量中存储的仅仅是地址，通过new关键字创建的对象，如Object, Array, Date，存储时放到堆里面
    • 可以理解为指针，间接寻址
