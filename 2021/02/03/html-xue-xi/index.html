Html学习

$1.1 网页

  • 网站：在因特网上根据一定规则，使用Html等制作的网页集合。
  • 网页：网站的一页，构成网站的基本要素，俗称html文件
  • Html：超文本标记语言，用来描述网页，两个方面理解：超越文本限制，超级链接文本

$1.2 浏览器

  • 常见浏览器：火狐，谷歌，Edge，Safari

  • 浏览器内核(渲染引擎)：负责读取网页信息，整理讯息，计算网页的显示方式并显示网页

    浏览器 内核
    IE Trident
    Fire Fox Gecko
    Safari WebKit
    Chrome Blink (WebKit分支)

    国内浏览器一般用WebKit或Blink内核。

$1.3 Web标准

W3C：万维网联盟，国际著名标准化组织

Web标准构成：

  • 结构(Structure):对网页元素进行整理分类​，主要指Html
  • 表现(Presentation):设置外观样式，主要指CSS
  • 行为(Behaviour):设置交互，主要指JavaScript

$1.4 Html标签

1. 语法规范

尖括号包含，一般成对出现。例：

2. 基本结构标签

  • html标签：页面中最大的标签，称为根标签

  • head标签：文档头部

  • title标签：网页标题

  • body标签：文档的所有内容

    <!DOCTYPE html>
<html lang = "zh-CN">
<head>
	<meta charset = "UTF-8"/>
	<title>第一个页面哦</title>
</head>

<body>
    
</body>
</html>

3. 编辑器

VScode，H5等

4. 骨架结构探索

  • <!DOCTYPE html>：告知浏览器采用哪种html版本显示网页，必须位于开头，是一个文档类型声明标签

  • <html lang = "en">：定义当前文档显示的语言。en为英语，zh-CN为中文

  • <meta charset = "UTF-8" />：定义字符集

5. 常用标签🔴

  • 标题标签：<h1> - <h6>

  • 段落标签：<p>

  • 换行标签：相比段落标签没有距离差。<br />

  • 文本格式化标签：

    • 加粗：<strong>，<b>
    • 倾斜：<em>，<i>
    • 删除线：<del>，<s>
    • 下划线：<ins>，<u>

  • 盒子标签：<div>单独占一行，<span>不换行

  • 🔴图像标签：<img src=""/>

    • src属性：指定图像文件，必须属性

    • alt属性：图像不能显示时的文本

    • title属性：鼠标悬浮时显示的文本

    • width,height属性：设置宽度高度

    • border属性：加边框，一般使用CSS

      注意点：不同属性不分先后顺序，之间空格分隔，键值对格式

  • 目录文件与根目录

  • 图像标签与路径

    1. 相对路径：下一级：`/`，上一级：`../`
2. 绝对路径：很少使用，电脑地址或者网络图片地址

  • 🔴超链接标签：<a href="跳转目标" target="目标窗口弹出方式">文本或图像</a>

    • target：默认_self，当前窗口覆盖；_blank则为新窗口打开

    • 链接分类

      1. 外部链接：http://www.qq.com

      2. 内部链接：在网站内部页面之间实现跳转

      3. 空连接：#

      4. 下载链接：href地址是一个文件或者压缩包，则会下载

      5. 网页元素的链接：图片视频等的超链接

      6. 锚点链接：定位到某个位置

        • <a href="#Name">Test</a>
        • <h3 id="Name">This</h3>

  • 注释标签

    <!--这是一条注释-->

    实际开发中快捷键：CTRL + /

  • 特殊字符

    字符 对应符号
    空格 &nbsp;
    小于 &lt;
    大于 &gt;

6. 常用标签进阶

  1. 表格标签

    <table> : 定义表格

    <tr>：定义表格一行

    <td>：定义表格单元格

    <th>：表头单元格标签（加粗居中显示）

  2. 表格属性(位于table标签中)

    align：对齐方式，有left，center，right三种

    border：边框大小，默认cellspacing为1，所以为双边框

    cellpadding：内容与单元格边框距离

    cellspacing：单元格之间距离

    width,height：表格宽度和高度

  3. 表格结构标签

    使结构更加清晰，无实际作用

    <thead>：定义表头，内部必须有tr

    <tbody>：定义表格主题部分

  4. 合并单元格

    • 跨行合并：rowspan，写到最上侧
    • 跨列合并：colspan，写到最左侧

  5. 🔴无序列表

    <ul>：无序列表主体，里面只能放<li>

    <li>：每一个选项，里面基本任何标签都可以放

    • a
    • b

  6. 有序列表

    <ol>：有序列表主体，里面只能放<li>

    <li>：每一个选项，里面基本任何标签都可以放

  7. 自定义列表

    <dl>：定义列表，只能包含<dt>和<dd>

    <dt>：定义项目

    <dd>：定义每一项

  8. 🔴表单标签

    主要是为了搜集用户信息，一个完整的表单通常由：表单域，表单控件，提示信息构成

    • 表单域：<form>，把其范围内的表单元素提交给服务器，常用属性(这些暂时不必了解，在网络编程会学到)：

      • action：接受表单数据的url地址
      • method：表单的提交方式，get或post
      • name：指定表单的名称，以区分多个表单域

    • 表单元素

      注意，以下三个标签都应该写在<form>中，并且有那么属性

      • input标签

        • type属性
          • text：普通文本
          • password：密码
          • radio：单选按钮
          • checkbox：复选框
          • submit：把表单域中数据发送给服务器，默认文字为提交，可使用value属性更改
          • reset：重置所有表单，默认文字为重置，可使用value属性更改
          • button：定义一个按钮，通常和JS搭配使用，默认文字为空，也可使用value属性更改
          • file：文件上传，例如上传头像
        • name属性 ：主要给后台人员使用，用于定义名字，配合radio可实现单选
        • value属性：定义属性值，主要用于后台使用，也可用于text显示默认文本
        • checked属性：主要用于单选按钮和多选按钮可以设置默认值
        • maxlength属性：规定输入框的最大字符
        • placeholder属性：与value在文本框类似，但是输入后会自动消失

      • label标签：为input标签定义标注

        ​ label标签的for属性和input标签的id属性对应

      • select标签

        ​ 内部使用<option>标签表示每一个元素

        • selected属性：默认选中

      • textarea标签

        • rows和cols属性：文本框的行数和列数
