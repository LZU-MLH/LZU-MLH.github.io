$1 Emmet语法

​ Emmet语法的前身是Zen Coding，它使用缩写，来提高Html/CSS的编写速度，VScode内部已经集成该语法。

$1.1 生成Html结构语法

  • 生成标签：div + tab键

  • 生成多个标签：div*10 + tab键

  • 生成父子级关系的标签：ul>li + tab键，div>span + tab键

  • 生成兄弟关系：div+p + tab键

  • 生成带类名或者id名标签：

    ​ p.one + tab键 <=> <p class="one"></p>

    ​ .two + tab键<=><div class="two"></div>

    ​ ul>li#three + tab键<=><ul><li id="three"></li></ul>

  • 生成顺序类标签：

    .demo$*2 + tab键<=><div class="demo1"></div> <div class="demo2"></div>

  • 带默认文字：

    div{乐饮过三觉}<=><div>乐饮过三觉</div>

$1.2 生成CSS样式语法

  • w100<=>weight: 100px;
  • ti2em<=>text-indent: 2em;
  • tac<=>text-align: center;

$1.3 快速格式化代码

VScode中：Shift + Alt + F

$2 复合选择器

  • 更准确，更高效的选择目标元素
  • 由两个或者多个基础选择器，通过不同方式组合而成
  • 常用的复合选择器：后代选择器，子选择器，并集选择器，伪类选择器

$2.1 后代选择器🔴

  • 语法：元素1 元素2 ...{样式声明}，例：ul li {color: red;}，表示选择ul里面的所有li标签，不论辈分
  • 注意点：
    • 元素之间空格相隔
    • 元素1是父级，元素2是子级，最终选择的是元素2
    • 元素2和以后的元素可以是儿子或孙子，只要是元素1的后代即可
    • 元素可以是任意基础选择器

$2.2 子选择器

  • 语法：元素1>元素2 {样式声明}，例如：div > p {color: red;}

  • 注意点：

    • 元素之间大于号间隔
    • 元素1是父级，元素2是子级，最终选择的是元素2
    • 元素2必须是亲儿子

$2.3 并集选择器

  • 语法：元素1,元素2,... {样式声明}，例如：div, p {color: red;}
  • 注意点：
    • 选择多级标签，让他们有相同样式，中间以逗号分隔，最后一个选择器不需要逗号
    • 元素可以是任意选择器，不限于基础选择器
    • 语法规范一般为逗号之后换行

$2.4 链接伪类选择器

  • 语法：元素:效果 {样式声明}，例如：a:hover {color: red;}

    • a:link：选择所有未被访问链接
    • a:visited：选择所有已经被访问的链接
    • a:hover：鼠标覆盖链接时的效果
    • a:active：选择活动的链接

  • 实际开发中一般按照LVHA的顺序书写

  • 一般只需要写a和a:hover

$2.5 focus伪类选择器

  • 语法：元素:效果 {样式声明}，例如，input:focus {样式声明}

  • 主要针对<input>表单元素

$3 元素显示模式

$3.1 基础知识

  • 元素显示模式就是元素(标签)以什么方式显示。比如<div>独占一行，<span>不换行

  • Html元素一般分为块元素和行内元素

  • 块元素特点

    • 自己独占一行
    • 高度，宽度，内外边距都可以设置
    • 宽度默认是容器(父级宽度)的100%
    • 是一个容器及盒子，里面可以放行内元素或者块级元素
    • 文字类的元素，例如<p>，<h1>，内部不能使用块级元素

  • 行内元素：<a>，<strong>， <b>等

    • 相邻行内元素在一行上，一行可以显示若干行内元素
    • 高，宽直接设置无效
    • 默认宽度就是它本身内容的宽度
    • 行内元素只能容纳文本或其他行内元素
    • 注意点：
      • 链接里面不能再放链接
      • 特殊情况<a>内可以放块级元素，但是需要转化

  • 行内块元素：<img>，<input>，<td>等

    • 一行可显示多个行内块元素，但之间会有空袭（行内元素特点）
    • 默认宽度就是它本身内容的宽度（行内元素特点）
    • 高度，行高，内外边距都可以控制（块级元素特点）

$3.2 显示模式的转化

  • 🔴行内元素转化为块级元素：增加链接<a>的触发范围

  • 块级元素转化为行内元素

  • 🔴行内元素转化为行内块元素

    <style>
    a {
        width: 150px;
        height: 50px;
        display: block;
    }
    div {
        display: inline;
    }
    span {
        width: 150px;
        height: 30px;
        display: inline-block;
    }
</style>

  • 小技巧：文字垂直居中：行高ling-height = 盒子高度，偏小则文字偏上，偏大则文字偏下

$4 CSS的背景

$4.1 背景颜色

​ background-color：默认为transparent透明，可以指定颜色，指定类型参考CSS_1的文本颜色三种类型。

$4.2 背景图片

​ background-image:url() | none

  • 常见于Logo，一些装饰性的小图片，超大的北京图片，优点是便于控制位置

$4.3 背景平铺

​ background-repeat:no-repeat | repeat | repeat-x | repeat-y

  • 默认repeat

$4.4 背景图片位置

​ background-position: x y;

​ x和y可以使用方位名词或者精确单位

  • 方位名词：top,bottom,center和left,right,center
    • 如果指定了两个，则前后顺序无关
    • 如果指定了一个，则第二个默认居中对齐
  • 精确单位：x坐标和y坐标，顺序固定，单位为px
    • 如果只指定了一个，那么该数值一定是x坐标，另一个默认垂直居中
  • 混合单位：x坐标和y坐标，顺序固定
    • 例background-position: 20px bottom;,background-position: center 40px;

$4.5 背景图像固定

background--attachment: scroll | fixed，默认scroll，以后做滚动视差会用到

$4.6 背景属性复合写法

没有特定书写顺序，一般书写顺序：

  • 背景颜色，背景图片，背景平铺，背景图像滚动，背景图片位置

  • 例：background: pink url(a.png) no-repeat fixed center top;

$4.7 背景色半透明

  • background: rgba(0, 0, 0, 0.3)，a=1为不透明，

  • 0.3可以简写为.3，把0省略

$5 CSS三大特性

  • 层叠性：相同的选择器设置相同的样式，此时一个样式会覆盖另一个样式，遵循就近原则(底层原则)
    • 哪个样式离结构近，就执行哪个样式
div {
	color: red;
    font-size: 18px;
}
div {
    color: green;
}
/*最终结果：color: green，font-size: 18px;*/

  • 继承性：子类会继承父类的样式，例如根据上个代码块的样式：<div><p>我继承了div</p></div>，则p标签会有color和font-size

    • 子类会继承text-,font-,line和color等

    • 行高的继承：行高可以设置为1.5，表示当前文字大小的1.5倍。

  • 优先级： 当一个元素指定多个选择器，就会产生优先级

    • 选择器相同，执行层叠行
    • 选择器不同，根据选择器权重执行
      • 权重由四位数字组成，不会有进位
      • 等级判断从左到右，如果某一位相同则判断下一位
      • 继承的权重为0

  • 权重叠加

    <style>
    /*权重: 0, 0, 0, 1*/
    li {
        color: red;
    }
    /*权重: 0, 0, 0, 2*/
    ul li {
        color: green;
    }
    /*权重: 0, 0, 1, 1*/
   	.nav li {
        color: green;
    }
</style>
<body>
    <ul class="nav">
        <li>哈哈</li>
    </ul>
</body>
