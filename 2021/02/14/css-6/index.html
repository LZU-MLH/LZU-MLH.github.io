CSS_6

Html5新特性

新的特性，新的表单，新的表单属性

1. 语义化标签

​ div对于搜索引擎来说，是没有语义的，所以导入了：

  • <header>：头部标签
  • <nav>：导航标签
  • <article>：内容标签
  • <section>：定义文档某个区域
  • <aside>:侧边栏标签
  • <footer>:尾部标签

注意点：

  • 针对搜索引擎
  • 可以使用多次
  • 在IE9中，需要转化为块级元素
  • 移动端常使用

$1.1 多媒体标签

$1.1.1 视频

  • mp4格式基本所有浏览器都支持

语法：<video src="文件地址" control="controls"></video>

常见属性：

  • autoplay="autoplay"：视频就绪自动播放
  • controls="controls"：显示播放控件，可用JS调整控件样式
  • width/height=像素值：设置播放器宽高
  • loop="loop"：播放完继续播放视频
  • preload="auto | none"：规定是否预先加载视频(如果有了autoplay，则忽视该属性)
  • muted="muted" ：静音播放
  • poster=imgurl：加载时的画面图片地址
  • src=url：视频地址

$1.1.2 音频

  • mp3格式基本所有浏览器都支持

语法：<audio src="文件地址" controls="controls"></video>

常见属性：

  • autoplay="autoplay"：视频就绪自动播放
  • controls="controls"：显示播放控件，可用JS调整控件样式
  • loop="loop"：播放完继续播放视频
  • src=url：音频地址

2 input表单

  • 新增类型：email | url | date | time | month | week | number | tel | search | color
  • 新增属性：
    • required="required"：必须填写内容
    • placeholder="提示文本"：表单的提示信息
    • autofocus="autofocus"：光标自动定位
    • autocomplete= off | on：重复打开网页时，是否记录历史输入内容，默认为on
    • multiple="multiple"：多选文件提交，常用于type="file"

CSS3新特性

1. CSS3现状

  • 有兼容性问题，IE9+才支持
  • 移动端支持更好
  • 正在不断改进中，且应用相对广泛

2. 新增选择器

$2.1 属性选择器

语法：

......
<style>
    /*必须是input，但同时要具有value属性*/
    input[value] {
        color: red;
    }
    /*必须是input，但同时类型要是text*/
    input[type="text"] {
        color: red;
    }
    div[class^="icon"] {
        color: red;
    }
</style>
......
<input type="text" value="请输入">
<input type="text">
<div class="icon1">图标1</div>
<div class="icon2">图标1</div>
<div class="icon3">图标1</div>
  • 类选择器，属性选择器，伪类选择器权重均为10

$2.2 结构伪类选择器

语法：

......
<style>
    /*选择ul里的第一个孩子，且必须是li*/
    ul li:first-child {
    	color: red;    
    }
    /*选择ul里的第二个,括号内可以是数字，表达式，关键字*/
    ul li:nth-child(2) {
    	color: red;    
    }
    /*选择ul里的所有奇数行，可用2n+1代替*/
    ul li:nth-child(odd) {
    	color: red;    
    }
    /*选择ul所有行，n从0开始递增*/
    ul li:nth-child(n) {
    	color: red;    
    }
</style>
......
<ul>
    <li>第一</li>
    <li>第二</li>
    <li>第三</li>
    <li>第四</li>
    <li>第无</li>
</ul>
  • div:nth-child(1)使用规则：先找孩子，再匹配E
    • 给每个盒子排列序号
    • 先看序号，匹配第一个元素
    • 如果第一个元素不是div，则样式无效，否则执行
  • div:nth-of-type(1)使用规则：先匹配E，再找孩子
    • 给指定元素盒子排列序号
    • 匹配完所有div元素
    • 如果匹配到第一个div，执行样式

$2.3 伪元素选择器

  • 利用CSS创建标签元素，而不需要Html标签，从而简化Html结构

语法：element::before | element::after {样式}

  • 本质：在父盒子内加入一个盒子，放到其他盒子前面或后面
  • 注意点
    • 创建的元素属于行内元素
    • 必须有content属性
    • 权重为1

$2.3.1 伪元素清除浮动

/*方法1*/
.clearfix::after {
	content: ""; /*必写内容*/
    display: block; /*必须为块级元素*/
    height: 0;	/*看不见盒子*/
    visibility: hidden;/*看不见盒子*/
    clear: both; /*清除浮动*/
}
/*方法2*/
.clearfix::before,
.clearfix::after {
    content: ""; /*必写内容*/
    display: table; /*转化为块级元素并且一行显示*/
}
.clearfix::after {
    clear: both; /*清除浮动*/
}

3. CSS3盒子模型

语法：box-sizing: content-box | border-box;

  • content-box：盒子大小为width + padding + border
  • border-box：盒子大小为width，因此border和padding就不会撑大盒子

4. 其他特性

  1. 图片模糊效果

    语法：filter: 函数();

    例：filter: blur(5px);

  2. calc函数

    语法：calc: 表达式;

    例：calc(100%- 30px);

5. 过渡属性

语法：transition: 属性 时间 曲线 开始时间

  • 属性：想要变化的CSS属性
  • 时间：持续时间，单位为秒
  • 曲线：默认为ease，类似物体自由下落，其他的有linear | ease-in | ease out | ease-in-out，可以省略
  • 开始时间：设置延迟出发时间，默认为0，单位为秒，可以省略

注意点：

  • 如果要写多个属性，以,间隔：transition: width 0.5s ease 0s, height .5s;
  • 所有属性同时变化可以使用all：transition: all 0.5s;
